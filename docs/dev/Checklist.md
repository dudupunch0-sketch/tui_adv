# escape from the office 개발 체크리스트

이 문서는 전체 개발 진행 상황을 추적하기 위한 체크리스트다.
체크박스는 실제 작업 완료 후 갱신한다.

## 상태 범례

- `[ ]` 아직 시작하지 않음
- `[x]` 완료
- `[~]` 진행 중으로 표시하고 싶으면 줄 끝에 `진행 중` 메모를 남긴다
- `[!]` 막힘이 있으면 줄 끝에 `BLOCKED:` 사유를 적는다
