# 플레이어 상태 설계

## 목적

플레이어 상태는 `escape from the office`의 선택 압박을 만든다.
상태는 단순한 숫자가 아니라, 어떤 선택지를 감당할 수 있는지, 무엇을 우선해야 하는지, 어떤 텍스트를 믿을 수 있는지를 결정한다.

## 기본 상태

| 상태 | 범위 | 방향 | 의미 |
|---|---:|---|---|
| 체력 | 0-100 | 높을수록 좋음 | 물리적 생존력 |
| 정신력 | 0-100 | 높을수록 좋음 | 괴현상, 공포, 회사적 광기를 견디는 힘 |
| 배터리 | 0-100 | 높을수록 좋음 | 휴대폰, 손전등, 사내망, 녹음/촬영, 지도 확인에 쓰는 전력 |
| 허기 | 0-100 | 낮을수록 좋음 | 시간이 지날수록 쌓이는 굶주림 위험 |
| 갈증 | 0-100 | 낮을수록 좋음 | 허기보다 빠르게 위험해지는 탈수 위험 |

## 기본 방향성

체력, 정신력, 배터리는 자원이다.

- 100에 가까울수록 안정적이다.
- 0에 가까울수록 위험하다.

허기와 갈증은 위험 게이지다.

- 0에 가까울수록 안정적이다.
- 100에 가까울수록 위험하다.
- 시간이 지날수록 증가한다.

TUI에서는 이 방향 차이를 명확히 보여줘야 한다.

예:

- 체력/정신력/배터리: 초록색이 많을수록 좋음
- 허기/갈증: 빨간색이 많을수록 나쁨

## 초기값 초안

1차 수직 슬라이스의 기본 시작값:

| 상태 | 시작값 |
|---|---:|
| 체력 | 100 |
| 정신력 | 90 |
| 배터리 | 65 |
| 허기 | 15 |
| 갈증 | 20 |

시작 상황은 야근 중 또는 늦은 시간 회사에 남아 있던 상태다.
따라서 체력과 정신력은 완벽하지 않고, 배터리는 이미 일부 소모된 상태로 시작한다.

## 판정 능력치

자원은 생존 압박을 만들고, 능력치는 선택지의 종류와 성공/실패 분기를 바꾼다.
`디스코 엘리시움`처럼 능력치가 높으면 같은 인카운터에서도 다른 해석/행동이 열린다.

1차 구현 능력치:

| 능력치 | 기본값 | 의미 |
|---|---:|---|
| logic | 2 | 문서, 규칙, 모순을 읽는 힘 |
| empathy | 2 | 메시지/흔적에서 사람의 의도를 읽는 힘 |
| volition | 2 | 공포와 회사적 압박을 버티는 힘 |
| composure | 2 | 들키지 않고 침착하게 행동하는 힘 |
| interface | 2 | 사내망, 단말기, 패널, 복합기를 다루는 힘 |
| physical | 2 | 문을 밀고, 도망치고, 버티는 힘 |

범위는 0-6이다.
능력치는 즉시 소모되는 자원이 아니라 선택지 조건과 2d6 판정 보정값으로 사용한다.

예:

- `interface >= 4`: 알림 패킷 지연 시간을 역추적하는 선택지 표시
- `logic >= 4`: 출력물의 문장 순서 오류를 알아차리는 선택지 표시
- `volition >= 5`: 정신력 피해가 큰 선택지의 실패 결과 완화 가능

## 턴 경과 규칙 초안

일반 턴 경과:

- 허기 +1
- 갈증 +2
- 위험도는 상황에 따라 변화
- 배터리는 기본적으로 감소하지 않음

배터리는 행동 비용으로 주로 소모한다.

배터리 소모 행동 예:

- 사내망 접속
- 휴대폰 플래시 사용
- 지도 확인
- 메시지 확인
- 사진 촬영
- 녹음
- 보안 패널 조작

## 임계치

### 체력

| 범위 | 상태 | 효과 |
|---:|---|---|
| 70-100 | 양호 | 페널티 없음 |
| 40-69 | 부상 | 위험 행동 성공률 소폭 감소 |
| 1-39 | 중상 | 이동/전투/도주 선택지 페널티 |
| 0 | 사망 | 실패 엔딩 |

### 정신력

| 범위 | 상태 | 효과 |
|---:|---|---|
| 70-100 | 침착 | 설명과 선택지를 정상적으로 인식 |
| 40-69 | 불안 | 괴현상 선택지 비용 증가 가능 |
| 1-39 | 붕괴 직전 | 텍스트 왜곡, 선택지 왜곡, 환청 로그 가능 |
| 0 | 정신 붕괴 | 실패 엔딩 또는 특수 코스믹 루트 |

정신력 낮음 연출 예:

정상:

1. 회의실 문을 연다
2. 예약 패널을 확인한다
3. 복도로 돌아간다

정신력 낮음:

1. 회의에 참석한다
2. 회의에 참석한다
3. 회의에 참석한다
4. 복도로 돌아간다?

- 정신력 낮음:
  - 정신력 1-39일 때 선택지 라벨 일부가 `▒`로 흐려질 수 있다.
  - 처음 임계 상태가 감지되면 `pressure_low_sanity_warning_seen` 플래그와 1회성 전조 로그가 붙는다.

### 배터리

| 범위 | 상태 | 효과 |
|---:|---|---|
| 60-100 | 충분 | 정보 행동 자유로움 |
| 30-59 | 부족 | 정보 행동을 신중히 선택해야 함 |
| 1-29 | 위험 | 사내망/조명/지도 행동 제한 |
| 0 | 방전 | 휴대폰 기반 선택지 대부분 불가 |

배터리 0은 즉시 게임 오버가 아니다.
하지만 어두운 구역, 사내망 단서, 현실 연결 루트를 크게 제한한다.

### 허기

| 범위 | 상태 | 효과 |
|---:|---|---|
| 0-39 | 정상 | 페널티 없음 |
| 40-69 | 배고픔 | 체력 회복량 감소 가능 |
| 70-89 | 굶주림 | 위험 행동 성공률 감소, 체력 서서히 감소 가능 |
| 90-100 | 아사 직전 | 이동/도주 페널티, 실패 위험 |

허기 감소 수단:

- 과자
- 컵라면
- 회의실에 남은 도시락
- 탕비실 간식
- 자판기 음식

### 갈증

| 범위 | 상태 | 효과 |
|---:|---|---|
| 0-29 | 정상 | 페널티 없음 |
| 30-59 | 목마름 | 정신력 회복량 감소 가능 |
| 60-79 | 탈수 | 체력/정신력 동시 압박, 환각 이벤트 가능 |
| 80-100 | 심각한 탈수 | 잘못된 선택지 표시, 실패 위험 |

구현된 1차 갈증 이벤트:

- 갈증 60 이상으로 탕비실에 있으면 `strange_water_dispenser` 인카운터가 후보가 된다.
- 갈증 60 이상이 처음 감지되면 `pressure_thirst_warning_seen` 플래그와 정수기 전조 로그가 1회만 붙는다.
- 정수기 환각은 갈증을 낮출 수 있지만 정신력 피해나 단서를 남긴다.
- `thirst_hallucination_seen` 플래그가 붙으면 같은 환각 인카운터가 반복 출현하지 않는다.

갈증 감소 수단:

- 생수
- 정수기 물
- 커피
- 자판기 음료
- 회의실에 남은 물

주의:

커피는 정신력을 조금 회복시키지만 갈증 회복량은 낮거나, 특정 상황에서는 불안/심박 상승 페널티를 줄 수 있다.

## 자원 상호작용

### 허기와 체력

허기가 높으면 체력 회복이 줄어든다.
굶주림 상태에서는 이동/도주/몸싸움 선택지가 불리해진다.

### 갈증과 정신력

갈증이 높으면 정신력 회복이 줄어든다.
탈수 상태에서는 회사 괴담과 코스믹 호러 이벤트가 더 자주 왜곡되어 보인다.

### 배터리와 정보

배터리는 “진실에 접근하는 힘”이다.
사내망, 메신저, 보안 패널, 녹음, 촬영은 대부분 배터리를 요구한다.

### 정신력과 선택지 신뢰성

정신력이 낮으면 선택지 텍스트가 반드시 정직하다고 볼 수 없다.
다만 플레이어가 지나치게 억울하게 느끼지 않도록, 왜곡에는 전조와 패턴이 있어야 한다.

## 실패 조건 초안

즉시 실패:

- 체력 0
- 정신력 0

조건부 실패 또는 강한 페널티:

- 허기 100
- 갈증 100
- 배터리 0 상태에서 암흑/격리 구역에 오래 머무름

## 1차 구현 기준

현재 도메인 모델은 다음 규칙을 먼저 고정한다.

- 체력 0: `health_depleted` 실패 상태
- 정신력 0: `sanity_depleted` 실패 상태
- 허기 100: 매 턴 체력 -2
- 갈증 100: 매 턴 체력 -4, 정신력 -2
- 배터리 비용 행동: `battery >= cost`일 때만 가능
- 정신력 1-39: 선택지 왜곡 후보 상태, 첫 감지 시 `pressure_low_sanity_warning_seen` 로그
- 갈증 60 이상: 환각 이벤트 후보 상태, 첫 감지 시 `pressure_thirst_warning_seen` 로그
- 능력치 기본값: `logic/empathy/volition/composure/interface/physical = 2`
- 능력치 범위: 0-6 clamp

허기/갈증 100은 그 자체로 즉시 게임 오버가 아니라, 지속 피해를 통해 체력/정신력 실패를 유도한다.

## 상태 변경 표기 원칙

모든 선택 결과는 가능한 한 명확히 보여준다.

예:

```text
체력 -8
정신력 -5
배터리 -3
갈증 -20
플래그 획득: printer_secret_started
```

단, 코스믹/히든 이벤트에서는 일부 결과를 즉시 숨기고 로그나 후속 이벤트로 암시할 수 있다.
